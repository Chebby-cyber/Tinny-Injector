# Tinny-Injector
this is an injector and pwm signal generator for automotive propouses, using cheap attiny85 and simple hardware with a really good results. 
can acchive rpms from 300 to almost 9000.
can simulate frecuencies from 5 to 140hz and a pwm from 0 to 100% dutty. 
the idea its simple emulate injector output to test, i try with 4 injectors high impedance and a drive by wire throtthle and works.
